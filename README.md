# Trello Clone

![](https://i.imgur.com/5kG5Vvg.png)

Check the [live demo](https://trello-clone-rts.vercel.app/).

## Stack

- TypeScript
- React w/Hooks
- State management: hooks (`useState`, `useReducer`) + Context API
- Styled Components

## How to run this project locally

1. clone the repo
2. `npm i`
3. `npm start`
